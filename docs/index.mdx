Dette er et undervisningsprojekt i formatet travel blog. Formaalet er at vise arbejde med indlaeg, indlaegsdetaljer, filtre, animationer og statistik ved brug af en moderne frontend-stack.

## Hurtig start

### Krav

- Node.js + npm

### Koer lokalt

```bash
npm install
npm run dev
```

### Build

```bash
npm run build
npm run preview
```

## Stack

- React + Vite
- Tailwind CSS
- React Router
- GSAP, Framer Motion
- Radix UI, Sonner
