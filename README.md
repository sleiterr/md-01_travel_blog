## About The Project

[![Product Name Screen Shot][product-screenshot]](https://md-01-travel-blog.vercel.app/)

This project is a travel blog built as a study assignment. It showcases a list of recent posts, detailed post pages with galleries, category filtering, and small statistics in the footer.

Key ideas:

- Highlight travel experiences in a clean, modern layout
- Provide easy filtering by category
- Add engagement cues like likes and comments counters

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

- Node.js + npm

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your-username/your-repo.git
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Start the development server
   ```sh
   npm run dev
   ```
